Mis Proyectos

Proyecto Apartado de Aulas con ASP.NET

He desarrollado una aplicación web robusta y eficiente utilizando ASP.NET, centrada en optimizar la gestión de aulas en mi universidad. Esta aplicación proporciona una plataforma fácil de usar para la reserva y seguimiento de aulas, mejorando la eficiencia y organización en el entorno académico. Desde la implementación de funcionalidades de reserva hasta la integración de una interfaz intuitiva, mi enfoque en el desarrollo web se refleja en esta solución que facilita la experiencia de estudiantes y profesores al interactuar con los recursos físicos de la universidad.

Calculadora Móvil con Xamarin.Forms

He desarrollado una intuitiva y funcional calculadora móvil utilizando Xamarin.Forms, demostrando mi experiencia en el desarrollo de aplicaciones multiplataforma. Esta aplicación combina una interfaz de usuario elegante con una lógica de cálculo precisa, brindando a los usuarios una herramienta fácil de usar para realizar operaciones matemáticas en sus dispositivos móviles. Desde el diseño de la interfaz hasta la implementación de las funciones, este proyecto refleja mi compromiso con la creación de aplicaciones móviles eficientes y atractivas.

Videojuego de plataforma con Unity

Como apasionado desarrollador de juegos, he creado un emocionante videojuego 2D utilizando Unity y programación en C#. Este proyecto representa mi habilidad para conceptualizar, diseñar y llevar a cabo experiencias interactivas envolventes. Desde la creación de gráficos hasta la implementación de mecánicas de juego, cada detalle ha sido cuidadosamente diseñado para brindar a los jugadores una experiencia única. Este juego no solo destaca mis habilidades técnicas, sino también mi creatividad y dedicación para ofrecer productos de alta calidad en el mundo del desarrollo de juegos.
